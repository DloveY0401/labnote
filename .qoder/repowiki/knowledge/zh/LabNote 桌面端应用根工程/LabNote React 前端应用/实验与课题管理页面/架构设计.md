本模块由四个独立的 React 函数组件构成，均位于 `src/pages` 下，作为路由级页面被上层路由表挂载：
- `ExperimentList.tsx`：实验清单页，通过 `db/queries` 并发拉取实验、课题、标签及实验-标签映射，使用 `useMemo` 按关键词/课题/标签/日期范围过滤，并以表格展示。
- `ExperimentEdit.tsx`：实验新建/编辑核心页，同时支持“保存为模板”和“导出实验细节”；通过 `modules/registry` 的 `STANDARD_MODULES` + `DEFAULT_LAYOUT` 驱动可拖拽排序、折叠/隐藏的标准与自定义模块布局，借助 `CustomModuleForm`、`ReactantTable`、`CatalystForm`、`SolventForm`、`TagSelector` 等子组件拼装表单；结构式绘制通过 `lazy(() => import('./StructureDraw'))` 懒加载，试剂选择走 `window.labnote.reagents.list()` IPC。
- `ProjectManager.tsx`：课题列表+内联新增/编辑表单，配合 `ConfirmDialog` 二次确认删除。
- `ProjectDetail.tsx`：课题详情与内联编辑，包含进度条可视化与关联实验计数。

依赖方向单一向下：页面 → `../components/*` / `../modules/*` / `../db/queries` / `../types` / `../utils/exportTemplates`，不反向引用其他页面；所有持久化操作统一经 `db/queries` 或 `window.labnote.*` IPC 桥接后端。
