- 对跨 iframe 的 Ketcher API 访问统一封装为 `waitForKetcher`/`getKetcher` 异步函数，并在 Electron 环境下先调用 `contentWindow.focus()` 再取 `ketcher` 实例。
- 所有分子结构修改都返回新的不可变 `Molecule` 对象（`addAtom`/`removeBond`/`updateAtom` 等），由上层通过 `onMoleculeChange(newMol)` 触发重绘，避免原地 mutation。
- Canvas 交互中需要持久化的临时状态（拖拽目标、首原子 ID、当前环模板）一律用 `useRef` 持有，避免 React 重渲染导致丢失。
- SMILES 解析将“词法/图构建”(`parseSmilesGraph`)与“属性计算”(`atomBondSum`/`smilesToAtomCounts`)拆分为独立函数，复用同一张 atom/bond 中间表示。
- 对外暴露的同步/异步分析入口成对出现：`analyzeSmilesSync` 仅做本地计算，`analyzeSmiles` 在此基础上追加网络查询，调用方按需选择。
