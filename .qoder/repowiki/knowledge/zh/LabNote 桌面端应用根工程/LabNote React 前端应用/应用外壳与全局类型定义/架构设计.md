模块由四个职责清晰的文件组成：
- `src/main.tsx` 是 Vite/React 18 的挂载点，通过 `ReactDOM.createRoot` 在 `#root` 下渲染，并用 `<HashRouter>` + `<React.StrictMode>` 包裹根组件。
- `src/App.tsx` 作为路由中枢，使用 `react-router-dom` 的 `<Routes>/<Route>` 声明式路由；常规页面统一嵌套在 `<Layout />` 布局内，而 `/schedule`、`/widget`、`/structure` 三个特殊路径各自提供独立 Layout（`ScheduleLayout`、`StructureLayout`），其中结构编辑器 `StructureDraw` 通过 `React.lazy` + `Suspense` 实现按需加载并带“加载中…”占位。
- `src/types.ts` 集中导出所有领域模型（Project、Experiment、Reactant、Catalyst、Solvent、Tag、Template、Task、Module* 等）以及 Create/Update Input 类型，并通过 `declare global { interface Window.labnote }` 暴露给运行时调用的 Electron IPC 桥接口（projects/experiments/tags/templates/reagents/modules/compound/tasks/widget 等命名空间），形成前后端契约的唯一来源。
- `src/styles/index.css` 基于 Tailwind CSS 3（`@tailwind base/components/utilities`），在 `@layer base` 中设置全局字体、字号、滚动条样式，在 `@layer components` 中封装 `.btn-*`、`.input-field`、`.card`、`.section-title` 等原子级 UI 类，供全应用复用。
依赖方向单向向下：main → App → pages/components，types 被各层共同消费，CSS 仅被 main 引入。
