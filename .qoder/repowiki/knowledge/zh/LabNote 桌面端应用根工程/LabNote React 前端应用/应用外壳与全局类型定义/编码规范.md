- 路由按功能分组：常规页面统一嵌套在 `<Route element={<Layout />}...>` 下，需要独立布局的路径（/schedule、/widget、/structure）单独声明并提供专属 Layout 函数。
- 重型页面（如 StructureDraw）一律使用 `React.lazy(() => import(...))` 配合 `<Suspense fallback=...>` 进行代码分割，fallback 采用居中提示文本。
- 领域模型以 `interface` 形式集中在 types.ts，输入类型通过 `Omit<..., 'id'|'experiment_id'>` 或 `Partial<CreateXxxInput>` 派生 Update 类型，保持 Create/Update 成对出现。
- Electron IPC 能力通过 `declare global { interface Window.labnote }` 扩展全局类型，每个子域（projects/experiments/tags/...）提供 list/get/create/update/delete 等 Promise API，前端只依赖此类型契约。
