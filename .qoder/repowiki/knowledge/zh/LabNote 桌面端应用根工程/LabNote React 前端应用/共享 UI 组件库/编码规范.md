- 表单组件统一采用受控模式：以 props 接收当前值，通过 `onChange`/`onSave` 回调向父级上报变更，不持有内部 state 作为唯一事实源。
- UI 元素通过 Tailwind 原子类组合构建，按钮统一使用 `btn-primary`/`btn-ghost`/`btn-danger`，输入框使用 `input-field`/`select-field` 等语义化类名。
- 标签选择器复用同一套 `PRESET_COLORS` 预设色板，并在新增/编辑时以圆形色块按钮供用户挑选颜色。
- 异步操作（创建/更新/删除标签）统一用 try/catch 包裹并静默吞掉错误，成功后触发 `onTagsRefresh` 回调让父组件刷新列表。
- 弹窗/下拉菜单通过本地 `useState` 控制显隐，点击外部区域关闭时使用 `useRef` + `document.addEventListener('mousedown', ...)` 实现点击外部关闭。
- 确认对话框采用固定定位覆盖层（`fixed inset-0 z-[10000]` + `bg-black/30`）+ 居中白卡结构，用于删除等危险操作的二次确认。
