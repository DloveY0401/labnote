本模块是纯展示与交互的无状态/受控组件集合，按职责平铺在 `src/components/` 下，无子目录分层：
- 布局层：`Layout.tsx` 通过 react-router 的 `<Outlet />` 包裹侧边栏与主内容区；`Sidebar.tsx` 负责导航。
- 数据输入层：`TaskForm.tsx`、`CatalystForm.tsx`、`SolventForm.tsx`、`ReactantTable.tsx` 以 props 接收受控值并通过回调（`onSave`/`onChange`）向上提交；其中 `TagSelector.tsx` 和 `TaskForm` 内的标签管理直接调用 `../db/queries` 的 `createTag/updateTag/deleteTag`，形成组件到持久化层的单向依赖。
- 过滤与展示层：`FilterBar.tsx` 聚合搜索/项目/标签/日期范围筛选；`TaskItem.tsx`、`LinkedTasks.tsx`、`StructureImage.tsx` 渲染具体业务实体。
- 通用 UI 层：`ConfirmDialog.tsx`、`Toast.tsx`（导出 `useToast` hook + `ToastContainer`）、`TagSelector.tsx` 作为可复用原子组件被上层组合使用。
依赖方向：组件 → `../types` 类型定义，部分组件 → `../db/queries`；组件之间仅通过 props 通信，无全局状态耦合。
