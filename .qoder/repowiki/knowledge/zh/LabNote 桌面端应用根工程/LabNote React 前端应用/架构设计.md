- 入口 main.tsx 通过 HashRouter + StrictMode 挂载 App，App.tsx 集中声明所有路由并按布局需求拆分为 Layout / ScheduleLayout / StructureLayout / WidgetPage 四种外壳。
- src/types.ts 定义全局 TypeScript 契约（Project、ExperimentDetail、Task、ModuleTemplate 等）并在 global.Window.labnote 上声明后端 IPC 方法签名，所有 pages 与 components 均按此契约调用 labnote.* API，形成“前端类型 → Window.labnote 接口”的统一数据通道。
- 子模块分层：pages/* 承载业务页面（experiments_projects），components/* 提供跨页面 UI（shared_components），modules/* 实现可插拔的实验模块模板系统，utils/* 支撑结构绘制（structure_drawing），由 App.tsx 统一编排。
- 结构绘制页 StructureDraw 使用 lazy() 异步加载并包裹在独立的全屏 StructureLayout 中，避免与主布局耦合。
