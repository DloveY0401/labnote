- IPC 通道名采用 `<domain>:<action>` 命名约定（如 `projects:list`、`experiments:create`、`widget:toggle`），并在 preload 中以相同命名空间组织 API。
- 写操作统一使用 `db.prepare(...).run()` 参数化语句，避免字符串拼接 SQL；涉及多表写入时使用 `db.transaction(fn)` 包裹以保证原子性。
- 更新类 handler 先 `SELECT *` 读取当前行，再对可选字段用 `data.field ?? current.field` 做就地合并，保持向后兼容。
- 对外暴露的 API 通过 `preload.ts` 中的 `LabNoteAPI` 接口进行类型契约，渲染进程只依赖该接口而非直接调用 ipc。
- 数据库 schema 变更通过 `PRAGMA table_info` 检测列是否存在后增量 `ALTER TABLE ADD COLUMN`，并以 `user_version` 占位预留未来版本迁移点。
