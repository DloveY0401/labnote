根目录作为 Electron 应用的入口（package.json 的 main 指向 dist-electron/main.js），通过 npm scripts 把两个子模块串联起来：`build:renderer` 用 Vite 把 react_frontend 编译到 dist/，`build:electron` 用 ts-node 把 electron_runtime 编译到 dist-electron/；`dev` 脚本同时启动 Vite dev server 和 Electron，由 electron_runtime 加载本地 http://localhost:5173 完成热重载。

- 构建产物约定：Vite 输出 `dist/**/*`，Electron 主进程输出 `dist-electron/**/*`，electron-builder.yml 的 files 字段统一打包这两处，并通过 asarUnpack 保留 better-sqlite3 与 Ketcher 等原生/大体积依赖。
- 运行时桥接：electron_runtime 负责 IPC、窗口与 SQLite 初始化，react_frontend 通过 HashRouter 组织页面，两者之间仅通过 Electron IPC 通信，无共享源码。
- 平台相关：electron-builder.yml 固定输出 Windows x64 NSIS 安装包，并配置国内 Electron 镜像加速下载。
